# 850 English OS — Domain Glossary

## 项目定位

基于 **C.K. Ogden 的 Basic English（基本英语）** 系统的英语学习 Web 应用。核心信念：850 个词 + 简单的语法结构 = 日常英语交流能力。目标用户是英语初学者，让他们不需要背一万个单词、不需要掌握复杂语法就能开口说英语。

## 核心概念

**Basic English 850**: Ogden 在 1930 年代创造的简化英语系统，850 个核心词汇覆盖日常表达所需。

**850 English OS**: 围绕这 850 词构建的学习系统，包含词库、每日练习、句型模式、以及简化器工具。

## Simplify / Rewrite 模块（规划中）

**定位**: 教学演示工具。用户输入一段复杂英文，系统用 850 词库内的简单词 + 简单句子结构输出简化版本。目的是向学习者展示 "复杂的东西用 850 词也能说清楚"，增强学习信心，同时作为理解练习的辅助工具。

## Patterns 模块

**定位**: 参考工具型。用户来查某个万能词有哪些 pattern、怎么用、在什么场合用。不是练习/测验模块。

**核心词**: 10 个万能动词 — get, make, take, put, give, go, come, have, be, do

**Pattern 信息维度**:
- `phrase` — 动介组合本身
- `zh` — 中文释义
- `literalBreakdown` — 字面拆解，解释为什么这个组合有这个意思（get + up → "到达上方" → 起床）
- `usageScenarios` — 使用场景标签（日常、工作、旅行等）
- `examples` — 多个例句（2-3 个），覆盖不同搭配
- `collocation` — 搭配结构（get up + 时间/早晚）
- `antonym` — 反义 pattern（get up ↔ go to bed）
- **不包含**: 近义/易混淆辨析（第一版跳过）

**导航与布局**:
- 顶部横向 pills 导航，点击切换单词，一次只显示一个词的内容
- 默认选中第一个词（get）
- 页面内分层: Hero 区 → Pattern 卡片网格 → 高级替代表
- Pattern 卡片采用网格布局（移动端 1 列，桌面端 2-3 列）
