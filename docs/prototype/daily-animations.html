在 Daily 页集成步骤

  1. 在 src/app/daily/page.tsx 顶部添加 import { motion, AnimatePresence } from "framer-motion"
  2. 卡片容器用 motion.div，设 initial/animate/variants 控制 stagger entrance
  3. 每张卡片用 motion.button，设 variants.hidden/visible 控制入场
  4. 翻转用 motion.div + style={{ transformStyle: "preserve-3d" }} + animate={{ rotateY: flipped ? 180 : 0 }}
  5. 正反面用 style={{ backfaceVisibility: "hidden" }}，背面加 transform: "rotateY(180deg)"
  6. 背景加 pointer-events-none 的 dust motes 容器
  7. 进度条改为 motion.div，animate={{ width }} 控制填充
  8. Shuffle 按钮用 rounded-[20px] + hover:text-valley-green
  9. 所有颜色引用 Tailwind v4 变量: bg-canvas-ice, text-adaline-ink, border-mist-gray, bg-valley-green 等
